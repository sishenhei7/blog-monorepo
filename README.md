# blog-monorepo

## feature

这个是 blog 的 monorepo，打算以楼层为中间纽带，主要分为下面几个模块：

- 独立的（业务）组件
- 楼层
- 渲染服务

### 独立的（业务）组件的功能

1. 各个组件独立打包，使用静态缓存
2. 各个组件与布局解耦，这样 blog 的布局可以随时更换，包括换肤
3. 由于是 blog，所以要能支持渲染 markdown 文件

### 楼层的功能

1. loading状态与骨架屏
2. 同步与异步楼层
3. 支持内嵌不同前端框架（后期做）

### 渲染服务的功能

1. 提供 ssr 渲染
2. 把各个组件根据 layout 组装到一起
3. 收集统计数据（uv、ip、监控）
